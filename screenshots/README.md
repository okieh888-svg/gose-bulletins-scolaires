# Captures d'écran

Ce dossier est destiné à recevoir les captures d'écran de démonstration
référencées depuis le README principal. Il est volontairement livré vide
dans ce dépôt (aucune capture n'a pu être prise dans l'environnement de
génération de ce prototype — voir la note en tête du README).

Captures à fournir (une par rôle, au minimum) :

- `01-login.png` — écran de connexion
- `02-dashboard-enseignant.png` — tableau de bord enseignant (mes classes)
- `03-saisie-notes.png` — écran de saisie des notes
- `04-apercu-bulletin.png` — aperçu HTML d'un bulletin (avant génération PDF)
- `05-dashboard-proviseur.png` — tableau de bord proviseur (file d'attente)
- `06-statistiques.png` — écran statistiques établissement
- `07-dashboard-eleve.png` — tableau de bord élève (bulletins publiés)
- `08-bulletin-pdf-fr.png` — rendu du bulletin PDF (français)
- `09-bulletin-pdf-ar.png` — rendu du bulletin PDF (arabe / RTL)
- `10-gestion-comptes.png` — écran de gestion des comptes (proviseur)

Nom de fichier libre tant que la référence est mise à jour dans le README.
